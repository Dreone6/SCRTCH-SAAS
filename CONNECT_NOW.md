## 🆘 Troubleshooting

**If connection fails:**

1. **Check tunnel is still active:**
   ```bash
   curl -s http://localhost:4040/api/tunnels | grep public_url
   ```

2. **Restart Expo if needed:**
   ```bash
   sudo supervisorctl restart expo
   ```

3. **Check new tunnel URL:**
   - The URL may change after restart
   - Check ngrok inspector: http://localhost:4040

**If you see "Unable to connect":**
- The tunnel URL might have changed
- Try the web preview URL first
- Restart Expo and check for new URL
